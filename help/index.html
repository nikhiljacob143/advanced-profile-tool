Advanced Profile Tool

Cross-section and long-section profiling for QGIS 3.44: generate section lines along an alignment, sample one or more DEM surfaces, compare surfaces, compute cut/fill areas and volumes, and export the results.

Full documentation: User guide — workflow, tab reference, chainage and offset conventions, volume methods, export formats and troubleshooting.

Quick start

  1. Load an alignment line layer (projected CRS) and one or more DEM rasters.
  2. Open the plugin, select the alignment feature and set the section interval and widths.
  3. Add the DEM surfaces to sample and choose the interpolation method.
  4. Generate: section lines are created and profiles sampled.
  5. Review profiles in the viewer, run surface comparison and volume calculations, then export.

Conventions at a glance

  • Offsets are signed: negative = left of the direction of increasing chainage.
  • Chainage display: 0+000 station format by default (CH / STA / plain also available).
  • NoData cells are never invented — they appear as gaps unless gap interpolation is enabled.
